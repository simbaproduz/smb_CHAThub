    YouTube

    YouTube

      Modo principal

      Compatibilidade real

      Funciona com live URL, channel URL, @handle ou channel ID. O quick start tenta leitura real da live publica sem pedir credencial do usuario.

      1. Cole a live, o canal, o @handle ou o channel ID no quick start.
      2. O monitor resolve o alvo publico e tenta entrar no chat real por compatibilidade.
      3. Se a live estiver offline, restrita ou sem chat, o app mostra isso com honestidade.
      Como funciona

      Modo avancado

      Fallback oficial

      Usa a API oficial do YouTube como fallback avancado. Cada usuario entra com sua propria credencial local. O pacote e o repositorio nao carregam credenciais privadas.

      1. Use isso so quando a compatibilidade nao bastar para a sua live.
      2. Salve primeiro o link da live ou um alvo resolvivel.
      3. Informe sua API key ou access token local para liberar o caminho oficial.
      Como conseguir a chave Credencial local do usuario
      Fallback oficial do YouTube

      Esta camada fica como fallback avancado do YouTube. A credencial continua salva apenas no arquivo local privado do usuario.

      Requisito minimo: primeiro salvar a live no quick start. Depois informar uma API key do YouTube Data API v3 ou um access token local.

      Campos avancados do YouTube

      Esta camada existe para troubleshooting, channel ID fixo, client ID/client secret e overrides locais. O fluxo principal do produto e a compatibilidade real acima.
